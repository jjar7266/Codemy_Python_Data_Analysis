# Codemy.com — Python Data Analysis
### Lessons, exercises, and practice files completed while following John Elder’s Data Analysis course

This repository contains all of my work from the Codemy.com **Python Data Analysis** course.
Each lesson is recreated with clean, organized code and expanded examples to reinforce core data‑analysis concepts using **Pandas**, **NumPy**, **Matplotlib**, and **Seaborn**.

---

## 📂 What’s Inside
- Individual lesson scripts with clear, Pythonic structure
- Data files (CSV samples, generated datasets, etc.) used throughout the course
- Teaching notes and summary blocks for future reference
- Additional examples beyond the course material
- Clean, readable scripts focused on learning, clarity, and reproducibility
- A consistent folder layout for easy navigation

---

## 📊 Topics Covered
- Working with **Pandas DataFrames**
- Importing, cleaning, and transforming data
- Filtering, sorting, grouping, and aggregating
- Handling missing data
- Reading/writing CSV files
- Using **NumPy** for numerical operations
- Data visualization with **Matplotlib**
- Statistical visualizations with **Seaborn**
- Histograms, bar charts, line charts, scatterplots, boxplots, KDE plots
- Real‑world data analysis workflows

---

## 🛠️ Environment & Tools
- **Python 3.x**
- **VS Code**
- Libraries:
  - `pandas`
  - `numpy`
  - `matplotlib`
  - `seaborn`

Some lessons (especially visualization lessons) are adapted for VS Code’s behavior with plots.
For example, Seaborn charts may appear in the **Plots** panel unless the backend is switched to `TkAgg`.
